About LMS

The project’s main aim is to build a management system more like a website which...

  1. Can create a computerized management system for a library
  2. Has the capability to issue books
  3. Has the capability to return books
  4. Has an administrator account
  5. Can manage book details through administrator account
  6. Tracks the books that users have issued
  7. Keeps the databases correct and up-to-date
  8. Can store all the book and user data in a proper manner
  9. Has a user account
  10. User can not access the admin section but admin can access the user section
Admin

In this library management system admin can perform the following operations after login...

  • Add Books
  • Search Books
  • Update or Edit Book Details
  • Delete Books
  • View Order Details
User

In this library management system user can perform the following operations after login...

  • See All Books
  • Place Order for Books
  • Return Books
  • View Date Info
  • See Delay date info and can see fine
